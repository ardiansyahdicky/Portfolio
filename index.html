Automatic Fish Feeder Prototype

  • An IoT-based automatic fish feeding system that ensures accurate feeding schedules (with offline support) and real-time water condition monitoring via a web dashboard.

Smart Enviro-Alert System Prototype

  • Designed as an Early Warning System (EWS) for sensitive areas like Server Rooms or Medical Storage to prevent damages caused by overheating.
  • Implemented an automated threshold-based algorithm to trigger Visual and Audio alarms (LED and Buzzer) if simulator values exceed safe limits.

Mini SCADA: Room and Security System

  • Adapted the industrial SCADA (Supervisory Control And Data Acquisition) concept to centrally monitor air quality and door security across two different rooms.
  • Utilized Grafana Cloud for analytical dashboards, InfluxDB Cloud for time-series databases, Google Sheets for data logging, and a Telegram Bot for remote control and alerts.
  • Based on the NodeMCU ESP8266 expanded with an ADS1115 ADC module to simultaneously read MQ-2, MQ-135, and DHT11 sensors with high precision.
